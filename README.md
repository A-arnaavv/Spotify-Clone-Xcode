# Spotify-Clone-Xcode

Online Music Player for iOS based on Spotify

API: https://developer.spotify.com/documentation/web-api/



# Features

-> Recommended tracks
-> Top artists
-> New releases
-> Playlists / Featured playlists
-> Explore categories
-> Search songs, artists, playlists, albums
-> Save / Delete songs
-> Shuffle
-> Repeat


# Stack

-> UIKit
-> Spotify Web API
